dipindahtangankan** untuk menggunakan dan memodifikasi source code Perangkat Lunak
untuk kebutuhan bisnis Pembeli sendiri, pada jumlah instalasi sesuai paket yang dibeli.

## 2. Yang BOLEH dilakukan
- Memasang dan menjalankan Perangkat Lunak untuk usaha milik Pembeli.
- Memodifikasi source code untuk kebutuhan internal Pembeli.
- Menggunakannya untuk melayani klien Pembeli (mis. sebagai jasa), selama source code
  tidak didistribusikan sebagai produk tersendiri.

## 3. Yang DILARANG

- Menghapus atau memalsukan atribusi/hak cipta penjual di dalam kode.

## 4. Kunci lisensi, update & dukungan
